Dlaczego OldOS?

Zbudowany dla wydajności

Każdy komponent jest dobrany tak, żeby wycisnąć maksimum ze starszego sprzętu.

⚡

Lekkie środowisko

Openbox zużywa tylko ~50 MB RAM. Żadnych zbędnych efektów, całe zasoby idą do gier.

🎮

Gotowy do grania

Wine i RetroArch preinstalowane. Uruchamiaj gry Windows i klasyki bez konfiguracji.

🐧

Debian Testing

Solidna baza z aktualnym oprogramowaniem i szerokim wsparciem sterowników.

💾

Minimalne wymagania

Działa na maszynach z 2–4 GB RAM i procesorami Intel Core 4. generacji.

🌐

NetworkManager

Prosta konfiguracja sieci Wi-Fi i Ethernet bez kombinowania w konsoli.

🖥️

Pełny pulpit

lxpanel, PCManFM i kompletne środowisko gotowe do codziennego użytku.

Specyfikacja

Co jest w środku?

Starannie dobrane komponenty — nic więcej, nic mniej.

Minimalne wymagania

  • Intel Core i5/i7 gen. 4
  • Minimum 2–4 GB RAM
  • Dysk HDD lub SSD

Środowisko graficzne

  • Openbox (window manager)
  • lxpanel (pasek zadań)
  • PCManFM (menedżer plików)
  • ~50 MB RAM w spoczynku

Gaming

  • Wine (gry Windows)
  • RetroArch (emulacja retro)
  • Obsługa joysticka i gamepada

System

  • Baza: Debian Testing
  • NetworkManager
  • nano, htop, curl, wget
  • ISO: ~1.1 GB

Instalacja

Jak zacząć?

Cztery kroki i masz gotowy system do grania.

Pobierz ISO

Ściągnij obraz dysku OldOS z naszej strony na SourceForge.

Nagraj na USB

Użyj Balena Etcher lub Rufus, żeby nagrać ISO na pendrive (min. 4 GB).

Uruchom z USB

Zrestartuj komputer i wybierz pendrive jako urządzenie startowe w BIOS/UEFI.

Zainstaluj

Postępuj zgodnie z instalatorem. Cały proces zajmuje kilkanaście minut.

# Nagrywanie ISO na USB przez terminal (Linux/macOS) sudo dd if=oldos-v1.0.iso of=/dev/sdX bs=4M status=progress # Lub użyj Balena Etcher — jest prostszy dla początkujących
