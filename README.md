##what OldOS is?
OldOS is a free Linux-based operating system designed for older computers.  
Give your hardware a second life and enjoy gaming without buying a new PC.

## Minimum Requirements

| Component | Requirement |
|-----------|------------|
| CPU | Intel or AMD (2015–2020 or newer) |
| RAM | Minimum 4 GB |
| Storage | 50 GB of free space |
| Disk Type | HDD or SSD (SSD recommended) |

## What’s Included?

- **Desktop Environment** — varies depending on edition:
  - KDE Plasma (main edition)
  - Xfce (lightweight edition)
  - Openbox + lxpanel (ultra-light edition, coming soon)

- **Taskbar** — lxpanel (where applicable)
- **File Manager** — PCManFM (or KDE alternatives in Plasma edition)
- **Gaming Support** — Wine + RetroArch preinstalled
- **Networking** — NetworkManager
- **Base System** — Inspired by Linux From Scratch (LFS)

## Editions

OldOS will be released in multiple editions:

- **OldOS KDE Edition** — full-featured, recommended version
- **OldOS Xfce Edition** — lightweight and fast
- **OldOS Openbox Edition** — ultra minimal (planned)

More editions will be introduced in future releases as the project evolves.
